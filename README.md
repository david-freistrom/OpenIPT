# OpenIPT

Meter Data Communication IP-Telemetry (Draft E-DIN 43863-4) Implementation with Node.js

!!! Using by your own risk !!!

## Install

```
$ npm install openipt --save
```

## Usage



## Developing

```
OpenIPT$ npm install
OpenIPT$ grunt build
OpenIPT$ grunt test

```
